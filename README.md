### Build Instructions
To build WAIVE-Plugins from source.

```
$ git clone --recursive https://github.com/ThunderboomRecords/WAIVE.git
```

#### Linux
Requires cmake

```
$ mkdir build
$ cd build
$ cmake ..
$ cmake --build . --config Release
```
